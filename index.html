Git Commands Cheat Sheet

git diff

Shows changes in files that are not yet committed.

git fetch

Downloads changes from the remote repository without merging them.

git pull

Fetches and merges changes from the remote repository into the current branch.

git stash

Temporarily saves uncommitted changes so you can work on something else.

git stash pop

Restores the most recent stash and removes it from the stash list.

git cherry-pick <hash>

Applies a specific commit from another branch to the current branch.
